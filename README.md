MIST
====

Mist is a dark-mode Windows utility dashboard for safe system review, RAM cleanup, hardware visibility, and restart-aware optimization workflows.

Mist V2 includes:

- Page-based navigation for Home, RAM Cleaner, PC Specs, Restart Optimizations, and Live Optimizations.
- A centralized Mist logo asset under `Assets/`.
- Live RAM and CPU statistics.
- Searchable optimization lists with synchronized selection state.
- Dark frosted-glass styling only.
